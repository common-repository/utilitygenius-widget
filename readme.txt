A Gutenberg block for displaying either an Ad or Full Page UtilityGenius widget.

== Description ==

A simple way to add UtilityGenius functionality to your WordPress website!

== Installation ==

1. Install the plugin
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Add the 'UtilityGenius Widget Block' Gutenberg block to your page.
